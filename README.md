🌍 TrueLife Foundation Website
🔗 Live Demo: https://truelifefoundation.vercel.app
📌 Project Overview
The TrueLife Foundation Website is a responsive and user-friendly web application designed to represent a non-profit organization focused on social welfare and community development.
This project aims to provide:
Information about the foundation
Details about its mission and vision
Donation and contact information
A clean, accessible, and modern UI

##🚀 Features
✅ Fully Responsive Design (Mobile + Desktop)
✅ Modern UI with Clean Layout
✅ About Us Section
✅ Mission & Vision Display
✅ Contact Section
✅ Deployed on Vercel

##🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Deployment: Vercel
Version Control: Git & GitHub
